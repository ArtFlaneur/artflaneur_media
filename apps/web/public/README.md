# Public Assets

## Логотип

Положите ваш файл логотипа в эту папку с одним из следующих имен:

- `logo.svg` (рекомендуется для векторной графики)
- `logo.png` (для растровых изображений)
- `logo.jpg` или `logo.jpeg`

### Требования к логотипу:

- **Формат**: SVG (предпочтительно) или PNG/JPG
- **Размер**: Высота автоматически установится 40px, ширина пропорциональна
- **Прозрачность**: Для PNG рекомендуется прозрачный фон
- **Качество**: Для PNG используйте разрешение 2x (например, 160px высота для Retina дисплеев)

### После добавления логотипа:

1. Если имя файла НЕ `logo.svg`, измените в `apps/web/components/Layout.tsx`:
   ```tsx
   src="/logo.svg"  // замените на ваше имя файла
   ```

2. Обновите страницу (Ctrl/Cmd + R) чтобы увидеть логотип

### Настройка размера логотипа:

В файле `apps/web/components/Layout.tsx` измените класс `h-10` на:
- `h-8` - меньше (32px)
- `h-12` - больше (48px)
- `h-16` - еще больше (64px)

Пример:
```tsx
<img 
  src="/logo.svg" 
  alt="Art Flaneur Logo" 
  className="h-12 w-auto"  // измените здесь
/>
```
